## Description
1. 설정이 쉽고 포트폴리오를 무료로 만들 수 있는 Bootstrap을 사용하였습니다.
2. 세부 정보를 추가하기위해 코드를 간단하게 편집하였습니다!
3. index.html 파일과 css 파일을 활용하여 만들었습니다.
